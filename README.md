# React - Movie Search & Trending App

A modern movie search application built using the TMDB public API. Users can search for movies by title, with trending movies surfaced alongside each search for easy discovery. The app uses Appwrite for backend services such as data persistence and user-related features, and is designed with a focus on performance, clean UI, and a smooth user experience.

## Key Features

- Real-time movie search using the TMDB public API
- Trending movies displayed on every search
- Appwrite integration for backend services and database management
- Responsive, user-friendly interface
- Optimized API calls and efficient state handling
- Clean, modular, and scalable code structure

## Tech Stack

- JavaScript / TypeScript
- React
- Appwrite
- TMDB Public API
